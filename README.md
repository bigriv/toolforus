# Nuxt3

## instaled

- create project
  - npx nuxi@latest init gameofus-nuxt
- typscript
  - npx nuxi typecheck
- sass
  - npm install sass
- pinia
  - npm install pinia
- axios
  - npm i axios
- prettier
  - npm install --save-dev prettier
- lint
  - npm install --save-dev @nuxtjs/eslint-config-typescript
- fabric
  - npm install @types/fabric
  - npm install fabric

## build

- npm run generate
